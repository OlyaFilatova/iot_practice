- [x] https://www.riverbankcomputing.com/static/Docs/PyQt6/introduction.html
    - PyQt6 is a set of Python bindings for Qt v6 from The Qt Company. 
    - Qt includes libraries and tools for
        - GUI
        - networking
        - threads
        - regex
        - SQL
        - SVG
        - OpenGL
        - XML
        - settings
        - location services
        - NFC/Bluetooth
        - web browsing
        - 3D animation
        - charts
        - 3D visualization 
    - additional libraries: PyQt6-3D, PyQt6-Charts, PyQt6-DataVisualization, PyQt6-Graphs, PyQt6-NetworkAuth, PyQt6-WebEngine 
    - Utility programs included:
        - `pyuic6` — converts Qt Designer GUIs to Python. 
        - `pylupdate6` — extracts translatable strings to `.ts` files for Qt Linguist. 
    - DBus support module (`dbus.mainloop.pyqt6`) available if `dbus-python` is installed - provides support for the GLib event loop.
    - QtDBus module provides Qt-style DBus interface.
- [x] https://www.riverbankcomputing.com/static/Docs/PyQt6/contributing.html
    - Users can contribute patches converting C++ references to Python.
    - Documentation written in reStructuredText and generated using Sphinx.
- [x] https://www.riverbankcomputing.com/static/Docs/PyQt6/eol_policy.html
    - When a Python version reaches it’s end-of-life, support for it will be removed in the next minor release of PyQt6
