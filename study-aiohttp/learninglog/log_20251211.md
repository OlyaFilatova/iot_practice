- [x] https://docs.aiohttp.org/en/stable/client_quickstart.html
    - Use `ClientSession()` and methods like `.get()`, `.post()`, `.put()`, `.delete()`, `.head()`, `.options()`, `.patch()`.
        - Use context manager or `await session.close()` after finished working with the session.
    - Use `base_url` for multiple requests to the same site.
    - Avoid creating a session per request; reuse sessions for efficiency.
    - Pass query parameters via `params` (dict, list of tuples, or string).
    - Supports multiple values per key using `MultiDict` or tuples.
    - param content is not encoded by library
    - URL canonicalization occurs automatically; 
        - can disable with URL construction and `encoded=True`.
            - Passing params overrides `encoded=True`
    - Response handling
        - Access status with `resp.status`.
    - To read text content - `await resp.text()`
        - can specify encoding.
    - To read binary content - `await resp.read()`.
    - To read JSON content - `await resp.json()`.
    - read(), json() and text() should be used carefully. All these methods load the whole response in memory
        - For large responses use `resp.content` (StreamReader).
        - It is not possible to use read(), json() and text() after explicit reading from content.
    - POST requests
        - to send form data - pass a dictionary to the data argument
        - to send text or binary - use data argument
        - to send json - use json argument
        - to send multipart-encoded file set data equal to `{'file': open('<file path>', 'rb')}`
        - example sending multipart-encoded file with filename and content_type
            ```python
            url = 'http://httpbin.org/post'
            data = aiohttp.FormData()
            data.add_field('file',
                        open('report.xls', 'rb'),
                        filename='report.xls',
                        content_type='application/vnd.ms-excel')

            await session.post(url, data=data)
            ```
        - to stream uploads simply provide a file-like object for the body or for a more controlled approach - use asynchronous generator
    - get and post requests can be chained because response.content is a StreamReader
    - to create WebSockets use session.ws_connect
    - By default aiohttp uses a total 300 seconds (5min) timeout
        - The value could be overridden by timeout parameter for the session
