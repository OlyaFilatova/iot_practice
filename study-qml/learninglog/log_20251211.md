    - QML language - declarative language for describing UIs via visual components and interactions.
        - JSON-like syntax with support for imperative JavaScript expressions combined with dynamic property bindings
    - The QML language and engine infrastructure is provided by the Qt Qml module.
    - Qt Quick - Standard library of QML types and functionality. Includes visual types, interactive types, animations, models/views, effects.
    - Qt Quick Controls module implements UI controls (e.g., buttons, menus, views) with built-in styles and custom style support.
